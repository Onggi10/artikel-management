# Article Manager

Frontend app untuk manajemen artikel dengan user dan admin role.

## Fitur
- Auth
- List, detail artikel
- CRUD artikel dan kategori (admin)
- Validasi, preview, pagination, search

## Teknologi
- Next.js App Router
- Tailwind + Shadcn UI
- Axios + Zod + React Hook Form
